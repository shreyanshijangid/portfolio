# Portfolio - Shreyanshi Jangid's Portfolio

A personal portfolio website featuring smooth GSAP scroll animations, an interactive canvas background, and a modern glassmorphism UI.

## Features

- **Interactive Background**: Dynamic canvas-based particles and gravity effects.
- **GSAP Animations**: High-performance scroll-triggered animations for a premium feel.
- **Glassmorphism UI**: Sleek, modern design aesthetics with translucent layered elements.
- **Responsive Layout**: Fully optimized for mobile, tablet, and desktop devices.

## Tech Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript, GSAP (ScrollTrigger, ScrollToPlugin)
- **Backend**: Node.js, Express.js

## Project Structure

```
Portfolio/
├── package.json         # Node.js dependencies and scripts
├── requirement.txt      # Project requirements
├── server.js            # Express server configuration
└── public/              # Static frontend assets
    ├── index.html       # Main HTML document
    ├── style.css        # Core stylesheet
    ├── main.js          # Client-side JavaScript and animations
    └── resume.pdf       # Downloadable resume
```

## Running Locally

1. Install the Node.js dependencies:
   ```bash
   npm install
   ```

2. Start the Express server:
   ```bash
   npm start
   ```

3. Open your web browser and navigate to the application URL (usually `http://localhost:3000` or the port shown in your terminal).
